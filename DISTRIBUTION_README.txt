Process Flow Builder - Distribution Package
================================================================================

This package contains everything needed to generate PowerPoint presentations
from simple text files describing your processes.

QUICK START:
------------

1. Place your process description file (.txt) in this folder
2. Optionally add my_power_logo.png for your logo
3. Optionally add key.txt for a Key box on the flowchart
4. Run ProcessFlowBuilder (double-click on Windows, or use command line)

The PowerPoint file will be created in the same folder with the same name
as your text file (e.g., my_process.txt â†’ my_process.pptx)

CONTENTS:
---------

- ProcessFlowBuilder (executable)
- USER_GUIDE.md (detailed instructions)
- example_process.txt (example file to get started)
- DISTRIBUTION_README.txt (this file)

REQUIREMENTS:
-------------

- Windows: Windows 10 or later
- Mac: macOS 10.13 or later
- Linux: Most modern distributions

No Python installation required!

- How to write process description files
- Formatting examples
- Troubleshooting tips
- Detailed feature explanations
